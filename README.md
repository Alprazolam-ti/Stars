## Technology Stack  
- React  
- TypeScript  
- Canvas API  
- CSS  

## Features Implemented  
- Scroll-driven transitions между секциями  
- Фиксированный fullscreen-layout  
- Плавная интерполяция параметров анимации  
- Кастомная анимационная петля через `requestAnimationFrame`  
- Оптимизированное обновление состояния скролла  
- Canvas-основанная анимация звёздного поля  
- Адаптивное поведение и поддержка pointer/touch событий
